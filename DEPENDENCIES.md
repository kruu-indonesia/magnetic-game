# Installed Dependencies

## ✅ Core Web Framework
- **React 18.3.1** - Modern UI library with hooks and concurrent features
- **React DOM 18.3.1** - React renderer for web browsers
- **Vite 5.4.10** - Next-generation frontend build tool with instant HMR

## 🎮 2D Gaming Library
- **Phaser 3.80.1** - Popular open-source HTML5 game framework
  - Features: Physics engines (Arcade, Matter.js), sprite handling, animations, input management
  - Perfect for 2D games: platformers, shooters, puzzle games, etc.

## 🎨 CSS & Design
- **Tailwind CSS 3.4.15** - Utility-first CSS framework
- **PostCSS 8.4.49** - CSS transformation tool
- **Autoprefixer 10.4.20** - Automatic vendor prefix addition

## 🛠️ Development Tools
- **@vitejs/plugin-react 4.3.3** - Official Vite plugin for React
- **ESLint 9.15.0** - JavaScript linter for code quality
- **TypeScript types** - Type definitions for React

## 🔄 Alternative Package Managers

You can use any of these package managers:

### npm (currently used)
```bash
npm install
npm run dev
```

### Bun (if installed)
```bash
bun install
bun run dev
```

### yarn
```bash
yarn install
yarn dev
```

### pnpm
```bash
pnpm install
pnpm dev
```

## 🚀 Quick Start

1. Dependencies are already installed!
2. Run `npm run dev` to start the development server
3. Open http://localhost:3000 in your browser
4. Click "Start Game" to see the Phaser demo
